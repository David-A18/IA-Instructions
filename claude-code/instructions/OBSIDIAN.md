# Obsidian + this vault

## Open as vault

1. Obsidian → **Open folder as vault** → select **this folder** (the directory that contains `_MOC.md` and `router.md`).
2. Pin **`_MOC.md`**; use **Local graph** on it or on any note.

## Why this helps

| Feature | Use |
|---------|-----|
| **Graph** | See how optimization ↔ MCP ↔ cookbook notes connect. |
| **Backlinks** | From any note, see what links here. |
| **Quick switcher** | `Ctrl/Cmd+O` — jump by title. |
| **Aliases** | `_MOC` has YAML `aliases` for faster search. |

## Wikilinks + Claude Code

- Format: `[[folder/note]]` (no `.md`). Obsidian resolves; in Claude Code open `folder/note.md`.
- **Router first** for tasks. **MOC** for graph navigation or after adding a new note (update `[[_MOC]]` links).

## Git

- `.obsidian/workspace.json` is gitignored (machine-specific). Re-open vault to regenerate.
